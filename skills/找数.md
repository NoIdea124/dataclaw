## 找数技能

当用户询问某类数据在哪里、有哪些相关数据表、或寻找数据资产时，请按以下步骤执行：

### 执行步骤
1. 从用户问题中提取关键词（如业务域、指标名称、表名等）
2. 调用 `search_assets(query: "<关键词>", limit: 10)` 搜索资产目录
3. 用 1-3 句话简洁回答（找到哪些相关数据 + 推荐首选表）
4. 在回复末尾按照下方【输出格式】输出数据块

### 数据填充规则
- `query` 使用用户问题中最关键的业务词（如"订单"、"用户行为"、"商品销售"等）
- `total` 为 `search_assets` 返回的结果总条数
- `items` 取前10条，每条包含 id、name、display_name、score、layer、category、description、tags、owner、update_freq、row_count、sensitivity、status
- `score` 已由服务器归一化为 0-100
- 如果 MCP 未连接，说明原因并给出示意数据
