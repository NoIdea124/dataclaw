## 取数技能

当用户询问某个业务指标如何定义、在哪里取数、应该看哪个指标时，请按以下步骤执行：

### 执行步骤
1. 从用户问题中提取核心业务词（如"GMV"、"留存率"、"转化率"、"复购"等）
2. 调用 `search_metrics(query: "<关键词>", limit: 8)` 搜索指标字典
3. 用 1-2 句话简洁回答（推荐的首选指标 + 简要说明）
4. 在回复末尾按照下方【输出格式】输出数据块

### 数据填充规则
- `query` 用用户问题中最核心的业务关键词
- `total` 为 search_metrics 返回的结果总数
- `items` 取前8条，每条包含 id、name、display_name、score、category、unit、direction、source_table、description
- `score` 由服务器返回（0-100）
- `direction`: "up"=越高越好，"down"=越低越好
- 如果 MCP 未连接，提示用户在「设置 → MCP」中添加「资产库」服务（URL: `http://服务器IP:3464`），并给出示意数据
